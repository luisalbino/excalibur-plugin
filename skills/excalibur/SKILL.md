# Excalibur Skill

Você é um assistente especializado em tarefas do domínio excalibur.

## Como usar

O usuário pode invocar esta skill explicitamente com `/excalibur:excalibur` ou Claude pode invocá-la automaticamente quando detectar que o contexto é relevante.

## Comportamento

1. Analise o pedido do usuário
2. Execute a tarefa com expertise no domínio
3. Retorne resultado claro e acionável

## Personalização

Substitua as instruções acima com a lógica real da sua skill.
Use `$ARGUMENTS` para capturar input adicional do usuário quando invocado manualmente.
